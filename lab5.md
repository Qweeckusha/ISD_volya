# Лабораторная работа №5 – Функциональные требования
## Информационная система представительства туристической фирмы

---

## Кратко о системе

Наша система помогает представительству турфирмы за рубежом:
- обрабатывать данные туристов из России
- оформлять визы и бронировать отели
- встречать группы в аэропорту
- организовывать экскурсии
- работать с грузом «шоп-туристов»
- считать деньги и готовить отчёты

---

## Основные функциональные требования

> FR – это сокращение от Functional Requirement (Функциональное требование)

### Модуль «Туристы и группы»
| ID | Требование | Описание |
|----|-----------|----------|
| FR-01 | Регистрация туриста | Система позволяет заносить данные туриста: ФИО, паспорт, пол, возраст, наличие детей, предпочтения по отелю |
| FR-02 | Категоризация туристов | Система автоматически относит туриста к категории: «отдых», «груз», «ребёнок» с учётом ограничений (дети не могут быть без родителей) |
| FR-03 | Формирование групп | Система группирует туристов по рейсам, датам, категориям для дальнейшей обработки |
| FR-04 | История посещений | Для каждого туриста сохраняется история: даты поездок, отели, экскурсии, сданный груз |

### Модуль «Визы и документы»
| ID | Требование | Описание |
|----|-----------|----------|
| FR-05 | Генерация визовых пакетов | Система автоматически формирует пакет документов для подачи в отдел эмиграции на основе данных туриста |
| FR-06 | Отслеживание статуса визы | Система позволяет отмечать статус: «подано», «получено», «отказ» с комментариями |
| FR-07 | Список для таможни | Формирование списков туристов для таможни (общий и по категориям) |

### Модуль «Размещение и отели»
| ID | Требование | Описание |
|----|-----------|----------|
| FR-08 | Бронирование номеров | Система позволяет бронировать номера в отелях на основе предпочтений туристов и доступности |
| FR-09 | Списки расселения | Автоматическое формирование списков: кто в каком отеле, кто в каком номере (с фильтрацией по категориям) |
| FR-10 | Статистика по отелям | Отчёт по загрузке отелей: количество номеров, проживающих, период |

### Модуль «Экскурсии»
| ID | Требование | Описание |
|----|-----------|----------|
| FR-11 | Каталог экскурсий | Система хранит расписание экскурсий, агентства-организаторы, цены, лимиты мест |
| FR-12 | Запись на экскурсии | Туристы (или менеджер) могут записываться на экскурсии, система проверяет доступность |
| FR-13 | Передача списков в агентства | Автоматическая генерация списков: «турист → экскурсия → агентство» для передачи партнёрам |
| FR-14 | Аналитика популярности | Система определяет самые популярные экскурсии и рейтинги агентств по качеству |

### Модуль «Груз и склад»
| ID | Требование | Описание |
|----|-----------|----------|
| FR-15 | Учёт груза туриста | Для «грузовых» туристов система ведёт весовую ведомость: маркировка, вес, упаковка, страховка |
| FR-16 | Формирование отгрузочных документов | Автоматическое создание ведомости отправки: кол-во мест, вес, стоимость, итоговая сумма |
| FR-17 | Статистика грузооборота | Отчёты: вес и количество груза за период, типы самолётов, доля разных видов груза |

### Модуль «Рейсы и аэропорт»
| ID | Требование | Описание |
|----|-----------|----------|
| FR-18 | Учёт рейсов | Система хранит данные о рейсах: дата, тип самолёта, кол-во мест, грузоподъёмность |
| FR-19 | Загрузка рейса | Расчёт и отображение загрузки: пассажиры, груз, объёмный вес |
| FR-20 | Список туристов по рейсу | Быстрый доступ к данным группы: состав, отели, груз, маркировка |

### Модуль «Финансы и отчётность»
| ID | Требование | Описание |
|----|-----------|----------|
| FR-21 | Учёт доходов и расходов | Система позволяет вносить статьи: отели, перевозки, экскурсии, визы, аэропорт, непредвиденные траты |
| FR-22 | Финансовый отчёт для головной фирмы | Автоматическая генерация полного отчёта по группе или периоду (с детализацией по категориям туристов) |
| FR-23 | Рентабельность представительства | Расчёт соотношения доходов и расходов, визуализация в виде графиков/процентов |
| FR-24 | Аналитика по типам туристов | Процентное соотношение «отдыхающие» / «шоп-туристы» в целом и по сезонам |

### Модуль «Запросы и отчёты» (общие)
| ID | Требование | Описание |
|----|-----------|----------|
| FR-25 | Поиск туриста | Быстрый поиск по ФИО/паспорту с выводом полной истории: поездки, отели, экскурсии, груз |
| FR-26 | Статистика посещаемости | Количество туристов, побывавших в стране за период (общее и по категориям) |
| FR-27 | Экспорт данных | Возможность выгружать списки и отчёты в форматы: PDF, Excel, CSV для печати или передачи |
